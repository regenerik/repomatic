Instrucciones de Uso de la API

Para utilizar esta API, sigue estos pasos:

  1. Envía una solicitud POST a la siguiente URL: http://repomatic.onrender/admin/ruta-futura.
  2. Proximamente estarán las instrucciones de cada ruta . Ej: 
    {
        "instrucciones": "futuras instrucciones aquí",

    }
